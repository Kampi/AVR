Detailed Description

Driver for Atmel AVR XMega USART module.

This contains the prototypes and definitions for the Atmel AVR XMega USART driver.

Author
Daniel Kampert
Bug:
  • No IRCOM mode
    • No 9 bit support
    • No synchronous mode
    • No Multi-Rrocessor Communication mode
